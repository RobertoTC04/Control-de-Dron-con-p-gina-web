Control-de-Dron-con-página-web
Descripción
Sistema de control inalámbrico para dirigir trayectorias de un dron con fines publicitarios. Interfaz web adaptable para PC y móvil desarrollada con HTML, CSS y JavaScript. Comunicación y procesamiento embebido implementados en C++ sobre ESP32. Se aplicaron conocimientos de electrónica y control para garantizar operación estable y precisa.

Funcionamiento (resumen)
La interfaz web (PC/móvil) envía comandos al ESP32

El ESP32 procesa comandos y transmite instrucciones al sistema de control

El sistema ejecuta trayectorias y movimientos definidos desde la web

Estado
Interfaz web y lógica de comunicación implementadas. La integración física final requiere adaptar el mecanismo de control al dron seleccionado.
